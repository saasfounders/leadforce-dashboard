# LeadForce Talent — Segment Profitability Matrix

Interactive GTM dashboard for segment prioritization and capital allocation.

## Live dashboard

👉 [Open dashboard](https://saasfounders.github.io/leadforce-dashboard/)

## What's inside

| Tab | Content |
|---|---|
| Segment matrix | Bubble chart — LTV/CAC vs TAM, quadrant overlays, alert rings |
| Monte Carlo | Interactive sliders — P10/P50/P90 sensitivity analysis |
| Resource allocation | Time & budget bars, ARR scenario model (Bear/Base/Bull) |
| Risk register | Risk table by segment + investor narrative checkpoints |
